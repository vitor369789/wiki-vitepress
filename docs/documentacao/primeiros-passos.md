# Primeiros Passos

Bem-vindo! Este guia vai ajudá-lo a começar rapidamente com o sistema.

## 📋 Pré-requisitos

Antes de começar, certifique-se de ter:

- [ ] Acesso ao sistema
- [ ] Credenciais de login
- [ ] Navegador atualizado (Chrome, Firefox, Safari ou Edge)

## 🚀 Início Rápido

### 1. Fazer Login

Acesse o sistema através do link fornecido e faça login com suas credenciais:

```
URL: https://seu-sistema.com.br
Usuário: seu-usuario
Senha: sua-senha
```

### 2. Configurar seu Perfil

Após o primeiro login, configure seu perfil:

1. Clique no ícone do usuário no canto superior direito
2. Selecione "Meu Perfil"
3. Preencha suas informações
4. Clique em "Salvar"

### 3. Explorar o Dashboard

O dashboard principal oferece uma visão geral de:

- **Atividades recentes** - Suas últimas ações no sistema
- **Notificações** - Alertas e avisos importantes
- **Atalhos rápidos** - Acesso rápido às funções mais usadas
- **Estatísticas** - Métricas e indicadores principais

## 💡 Dicas Úteis

::: tip Atalhos de Teclado
- `Ctrl + K` - Busca rápida
- `Ctrl + S` - Salvar
- `Esc` - Fechar modais
:::

::: info Primeiro Acesso
No primeiro acesso, você será solicitado a alterar sua senha padrão por segurança.
:::

## ❓ Dúvidas Frequentes

**Como recuperar minha senha?**
Clique em "Esqueci minha senha" na tela de login e siga as instruções.

**Posso usar em dispositivos móveis?**
Sim! O sistema é responsivo e funciona em smartphones e tablets.
